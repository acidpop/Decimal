# Decimal
C++ 개발자를 위한 초정밀 소수 클래스

## 소개
static(kmc7468)이 부동 소수점 방식을 이용한 소수 처리 방식은 완벽하지 못하다는 것을 깨닫고 대한민국의 초등학교에서 가르치는 세로셈 방식에서 영감을 얻어
정수 부분은 64비트(8바이트) 영역까지 (약 -900경~+900경 정도), 소수점 이하 부분은 무한하게 표현할 수 있도록 만든 C++ 전용 소수 클래스입니다.
